# 비밀번호 발음하기

## 느낀점
- 단순 구현문제

## 흐름
1. 각 문자를 순회하며 3가지 조건 체크
2. 모음 포함 여부 플래그로 관리
3. 이전 2개 문자를 저장해서 3연속 모음/자음 체크
4. 같은 글자 연속은 직전 문자와 비교 (ee, oo 예외 처리)

## 코드
```python
from collections import deque
import sys
input = sys.stdin.readline
moum = {'a', 'e', 'i', 'o', 'u'}
        
while True:
    inputStr = input().strip()
    before_1 = ' '
    before_2 = ' '
    findMoum = False
    inval = False
    if(inputStr == "end"):
        break
    for i in range(len(inputStr)):
        c = inputStr[i]
        
        # 모음 하나 이상
        if(c in moum):
            findMoum = True
        # 연속
        if(i > 1 and before_1 in moum and before_2 in moum and c in moum):
            inval = True
            break
        if(i > 1 and not before_1 in moum and not before_2 in moum and not c in moum):
            inval = True
            break
        if(c != 'o' and c != 'e'):
            if(before_1 == c):
                inval = True
                break
        before_2 = before_1
        before_1 = c
    if(not findMoum):
        inval = True
    if(inval):
        print("<"+inputStr+"> is not acceptable.")
    else:
        print("<"+inputStr+"> is acceptable.")
```

## AI 코드 리뷰

### 개선할 점

1. **불필요한 import 제거**: `deque`를 사용하지 않음

2. **변수명 개선**: `inputStr` → `pw`, `findMoum` → `has_vowel`, `inval` → `invalid`, `before_1/2` → `prev1/2`

3. **f-string 활용**: 문자열 연결 대신 f-string이 더 깔끔

4. **연속 글자 체크 로직 간소화**: `c not in {'e', 'o'}`로 묶을 수 있음

5. **모음/자음 판별 함수화**: 반복되는 `in moum` 체크를 함수로 분리하면 가독성 향상

## 수정된 코드
```python
import sys
input = sys.stdin.readline

VOWELS = {'a', 'e', 'i', 'o', 'u'}

def is_vowel(c):
    return c in VOWELS

while True:
    pw = input().strip()
    if pw == "end":
        break
    
    has_vowel = False
    invalid = False
    prev1, prev2 = ' ', ' '
    
    for c in pw:
        if is_vowel(c):
            has_vowel = True
        
        # 모음 또는 자음 3연속 체크
        if prev2 != ' ':
            if is_vowel(prev2) == is_vowel(prev1) == is_vowel(c):
                invalid = True
                break
        
        # 같은 글자 연속 체크 (ee, oo 제외)
        if c == prev1 and c not in {'e', 'o'}:
            invalid = True
            break
        
        prev2, prev1 = prev1, c
    
    if not has_vowel:
        invalid = True
    
    result = "acceptable" if not invalid else "not acceptable"
    print(f"<{pw}> is {result}.")
```

### 개선 사항 요약
- 불필요한 import 제거
- 변수명을 의미 있게 변경
- `is_vowel()` 함수로 모음 판별 로직 분리
- 3연속 체크를 `is_vowel()` 비교로 단순화
- f-string으로 출력 포맷 정리
